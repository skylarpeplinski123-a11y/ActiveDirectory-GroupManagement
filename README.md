# ActiveDirectory-GroupManagement

## Overview
This lab demonstrates how to create a **new security group** in **Active Directory Users and Computers (ADUC)**.

---

## Tools Used
- Windows Server (Domain Controller)
- Active Directory Users and Computers (ADUC)

---

## Steps Performed

1. Opened **Active Directory Users and Computers**
2. Navigated to the correct **Organizational Unit (OU)**
3. Right-clicked the OU → **New → Group**
4. Entered the **group name**
5. Left settings as:
   - Group scope: **Global**
   - Group type: **Security**
6. Clicked **OK** to create the group
7. Confirmed the group appeared in the OU

---

## Verification

- Confirmed the group shows in the correct folder  
- Opened group properties to ensure it was created successfully
